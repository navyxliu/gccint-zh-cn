22.2 为垃圾收集器标记Roots

除了追踪类型信息，类型系统还需要定位全局变量（roots），作为垃圾搜集器开始的地方。Roots必须按照下面的语法之一来声明：

  • extern GTY(([options])) type name;
  • static GTY(([options])) type name;
语法
  • GTY(([options])) type name;
是不被接受的。对于这样的变量，应该在某个头文件中存在一个extern声明——可以在那里标记，而不要在它的定义中标记。或者，如果变量只在一个文件中使用，则将其定义为static。
