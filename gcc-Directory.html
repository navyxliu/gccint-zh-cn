6.3 gcc 子目录

gcc 目录包含了GCC的许多C源文件，用于配置和构建过程的其它文件，以及文档和测试包子目录。GCC的源文件在一个单独的章节里进行介绍。参见编译器的Passes和相关文件。

  • Subdirectories: gcc 的子目录。
  • Configuration: 配置过程，以及所使用的文件。
  • Build: gcc 目录下的构建系统。
  • Makefile: gcc/Makefile 中的目标。
  • Library Files: gcc/ 下的库源文件和头文件。
  • Headers: GCC安装的头文件。
  • Documentation: 构建GCC中的文档。
  • Front End: 语言前端剖析。
  • Back End: 目标后端剖析。
