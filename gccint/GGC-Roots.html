22.2 Marking Roots for the Garbage Collector

除了了解类型信息，类型机构（type machinery）还需要定位全局变量 （roots），作为垃圾搜集器开始的地方。Roots必须按照下面的语法之一 来声明：

  • extern GTY(([options])) type name;
  • static GTY(([options])) type name;
语法
  • GTY(([options])) type name;
是 不 被接受的。对于这样的变量，应该在某个头文件中存在一个 extern 声明——可以在那里标记，而不要在它的定义中标记。或者， 如果变量只在一个文件中使用，则将其定义为 static。
